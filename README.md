# DistillBoard

DistillBoard is a Chrome extension designed for AI chats like ChatGPT, Claude, and Gemini.

It helps users collect, pin, rearrange, and export the most important parts of long conversations, so key insights do not get lost inside endless chat threads.

## Features

- Save important snippets from AI conversations
- Organize them in a side panel
- Rearrange and refine what matters most
- Export a condensed board of key information

## Privacy

DistillBoard processes page content only to let the user select and organize the information they want to keep.
